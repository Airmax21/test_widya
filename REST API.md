# REST API Documentation

## Authentication

### Register User

Registrasi User ke sistem.

- **Endpoint:** `/auth/register`
- **Method:** `POST`
- **Request Body:**

  ```json
  {
    "name": "Iqbal",
    "email": "iqbal.agyan@gmail.com",
    "gender": "laki",
    "password": "test2123"
  }
